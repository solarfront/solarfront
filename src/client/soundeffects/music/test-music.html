About the Music Engine

This music engine uses the Web Audio API to create musical compositions procedurally. It can recreate songs, play individual notes, chords, and full musical arrangements using oscillators and precise timing.

Features:

  • Musical note generation with proper frequencies
  • Chord progressions and harmonies
  • ADSR envelopes for realistic note shaping
  • Support for different octaves and wave types
  • Timing system for complex musical arrangements
